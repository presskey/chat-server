# Simple Chat Server

Minimal Rails application with [Faye](http://faye.jcoglan.com) middleware and [Thin](http://code.macournoyer.com/thin/) as default web server.

## Installation
```
$ git clone git@github.com:presskey/chat-server.git
$ cd chat-server/
$ bundle install
```

## Usage
```
$ rails s
```
